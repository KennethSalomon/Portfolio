# Kenneth Studio Portfolio 🚀

Bienvenue sur le dépôt de mon portfolio professionnel. Ce site est une vitrine immersive mettant en avant mes compétences en design et développement web/mobile.

## ✨ Caractéristiques
- **Défilement Fluide :** Optimisé avec [Lenis](https://github.com/darkroomroom/lenis) pour une réactivité instantanée.
- **Animations Immersives :** Propulsé par [GSAP](https://gsap.com/) et ScrollTrigger.
- **Design Responsif :** Interface "perfect" adaptée à tous les écrans (mobile, tablette, desktop).
- **Glassmorphisme :** Esthétique moderne utilisant des effets de flou et de transparence.

## 🛠️ Technologies
- **Frontend :** HTML5, CSS3, JavaScript (ES6+)
- **Librairies :** GSAP, Lenis, FontAwesome
- **Fonts :** Instrument Serif, Inter

## 🚀 Installation locale
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/KennethSalomon/Portfolio.git
   ```
2. Ouvrez `index.html` (auparavant `fiche.html`) dans votre navigateur.

## 📝 Note de Déploiement
Ce projet a été configuré pour GitHub Pages. **Important :** Renommez votre fichier `fiche.html` en `index.html` à la racine du projet avant de pusher.
